# 🏥 Hospital Management System (JSP & Servlet)

A Hospital Management System developed using **Java, JSP, Servlets, JDBC, and MySQL**. This project follows the **MVC (Model-View-Controller)** architecture and provides a simple, user-friendly interface for managing hospital records. It includes complete **CRUD (Create, Read, Update, Delete)** operations for different hospital modules.

---

## 📌 Features

- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Management
- 🏢 Department Management
- 📅 Appointment Management
- 💳 Billing Management
- 📊 Doctor Workload Report
- 📈 Recent Activity Dashboard
- ✏️ Update Records
- ❌ Delete Records
- 🔍 Search & View Records
- 🗃️ Complete CRUD Operations

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Hospital-Management-System.git
```

2. Import the project into Eclipse IDE.

3. Create a MySQL database.

4. Import the `hospital.sql` file into MySQL.

5. Update database credentials in the `DBUtil.java` file.

6. Configure Apache Tomcat Server.

7. Run the project.

8. Open your browser and visit:

```
http://localhost:8080/HospitalManagementSystem/dashboard
```
